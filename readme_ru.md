# Публичная информация по созданию ботов для GrandGames Arena

## Общая информация

Любой желающий может создавать ботов для игры в настольные игры на сайте GrandGames Arena используя приведенный SDK и запускать их на собственном ПК или VPS.

SDK представляет собой js скрипт и TypeScript интерфейс, позволяющий выполнять подключения к платформе, отправлять ходы, получать состояние досок, отправлять сообщения и т.д.

С его помощью можно реализовать функционал бота как для Node.js, так и для браузера. Для обоих окружений есть библиотеки для подключения с одинаковым интерфейсом. Разница лишь в том, что внутри библиотеки для браузера использует WebSockets из стандартного WebAPI, а для Node.js - пакет ws с npm.

Непосредственно движок бота может быть реализован на любых языках. К примеру, можно скомпилировать движок бота в исполняемый файл, а на node реализовать лишь обертку на sdk и подключать движок вашего бота через child_process, либо в WASM для веб-весрии SDK. Рекомендуемая минимальная версия Node - 24 LTS.

Ссылки на проекты, релизуйющие работу с данным SDK можно найти в файле [samples.md](samples.md)

## Порядок действия

1. Зарегистрировать на https://arena.grandgames.net/ новый аккаунт (обязательно оканчивать на Bot, с другим именем аккаунты могут быть забанены при использовании этого SDK), к примеру YourBot
2. Получить токен для авторизации (в настройках профилях есть кнопка)
3. Реализовать бота с использованием SDK arena-bot-node-sdk.js или arena-bot-web-sdk.js
   Каждый скрипт представляетс собой esm модуль, экспортирующий по-умолчанию класс, реализующий интерфейс описанный в IBotSDK.ts

В первую очередь сервис GrandGames Arena предназначен для игры людей.
Наша мотивация заключается в первую очередь в том, чтобы дать возможность поиграть людям, когда нет других игроков и соответсвенно сделать сервис более интересным и популярным. Во вторую очередь - предоставить увлеченным программированием и настольными играми людям, проверить свой скилл в разработке алгоритмов для игры.

В боте специально нет функционала создания столов. Бот может играть только на столах, предоставленных платформой. При подключении бота функционал платформы сам создает столы для игр, в которые ваш бот может играть и ожидает подключения игроков. Если игроков долго нет, платформа может организовывать матчи ботов друг с другом для определения реального рейтинга, проверки работоспособности ботов.

У каждого бота, как и у обычных игроков, есть свой ELO рейтинг для каждой игры.

В будущем, если найдется достаточное количество энтузиастов, мы планируем добавить турнирную систему и возможность соревноваться ботам друг с другом. Если вам это интересно, ставьте Star. Если вы реализовали и запустили своего бота с открытым исхожным кодом, можете редактировать Samples.md добаввив ссылку и создать Pull Request.
